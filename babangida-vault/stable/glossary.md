# Глоссарий — ubiquitous language

Единый язык babangida: что значит термин в продукте и в коде. Описываем _термины_,
а не _типы_ — структуры (`Invite`, `Profile`, …) и инварианты живут в крейте
`domain` (ADR-0003). Здесь — смысл и границы понятия, одной строкой.

Каркас на старте. Термины наполняются по мере появления домена (с Prompt 1).
Формат записи: **термин** — краткое определение; при необходимости связь с ADR.

## Термины

- **invite** — приглашение, единственный путь регистрации. Правила выдачи — ADR-0005.
- **verified** — верифицированный юзер; открывает маркет, музыку, open API (ADR-0010).
- **casual** — обычный (неверифицированный) юзер: лента, профиль, переписка.
- **public** — публичная сущность/паблик (значение уточняется в social/messaging контексте).
- **skin** — _TBD_ (термин из домена, уточнить при моделировании).
- **underground** — культурный контур комьюнити; задаёт тон продукта и модерации.
- **subculture** — субкультура профиля (underground / casual / skin / hiphop / …).
- **handle** — уникальный @-идентификатор юзера.
- **cooldown** — пауза между выдачами инвайтов (~12ч, ADR-0005).
- **quota** — лимит активных инвайтов на юзера (2; админ — без лимита, ADR-0005).
