# ADR-0003: Крейт domain — source of truth для типов и инвариантов

## Status

Accepted — 2026-06-25

## Context

Типы, модели и инварианты домена (инвайты, верификация, юзеры) нужно где-то фиксировать так, чтобы они не расходились между кодом и документацией. В соло/маленькой команде дублирование быстро гниёт: vault и код разъезжаются, и непонятно, где правда. Нужен один источник на факт.

## Options considered

- **Дублировать типы в vault/глоссарии.** Плюс: можно читать модель в markdown без захода в код. Минус: два источника, которые гарантированно разъедутся. Глоссарий начнёт врать, а проверить это нечем — нет компилятора для markdown.
- **Генерить типы из схемы БД.** Плюс: схема и типы синхронны. Минус: БД диктует домену форму, а это инверсия зависимостей — `domain` не должен знать про `sqlx`/Postgres. Инварианты (кулдаун инвайтов, лимит на 2 активных) в схему не выразить.
- **OpenAPI-first, генерить из контракта.** Плюс: контракт API первичен, клиенты получают типы бесплатно. Минус: API — это транспортный слой, не домен; контракт описывает наружу, а инварианты живут глубже. Получили бы тот же разрыв, только источником стал бы yaml вместо markdown.
- **`domain`-в-коде как истина (выбрано).** Типы и инварианты — это Rust-код в `domain`, компилятор их проверяет, остальные слои зависят от них внутрь (см. [[decisions/0002-monorepo-cargo-workspace]]).

## Decision

Каноническим источником типов, моделей и инвариантов является код крейта `domain`. Vault документирует решения и контекст (ADR, glossary, patterns), но не дублирует типы — это принцип «один source of truth на факт». Выбрано, потому что только код проверяется компилятором и не может тихо разойтись с реальностью.

## Consequences

Получаем: единственное место правды, инварианты под защитой типов и тестов, никакого дрейфа между доком и кодом. Схема БД и контракты API становятся производными от `domain`, а не наоборот.

Теряем: документация типов — это теперь чтение кода, а не markdown. Тот, кто захочет узнать форму `Invite`, идёт в `domain`, а не в глоссарий. Глоссарий в vault остаётся для терминов и смысла, но не для полей и сигнатур. Это сознательный размен: чуть выше порог входа в обмен на отсутствие вранья.
