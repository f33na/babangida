# AGENTS.md — `domain`

Перед работой прочитай `../babangida-vault/COMMON.md`. Релевантные ADR: 0003, 0004,
0005, 0010. Это source of truth для типов и инвариантов — vault их не дублирует.

## Роль

Доменное ядро: value objects, агрегаты, доменные события, порты (repository-трейты),
спецификации. Чистый Rust, без I/O.

## Границы

- Зависит ТОЛЬКО на `babangida-shared`. Никаких axum/sqlx/leptos/dioxus/HTTP/БД.
- Инвариант — в типе: парсинг-в-конструкторе, невалидные состояния непредставимы.
- Правила (квота/кулдаун инвайта, ADR-0005) живут здесь, а не в хендлерах.
