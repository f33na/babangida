# AGENTS.md — `api`

Перед работой прочитай `../babangida-vault/COMMON.md`. Релевантные ADR: 0004, 0005.

## Роль

HTTP-слой на Axum: роутинг, хендлеры, сериализация. Точка композиции зависимостей
(здесь собираются адаптеры из `infrastructure` под порты `application`).

## Границы

- Зависит на `application`, `domain`, `shared` (+ axum/tower). Композиция — только тут.
- Доменные правила вызывает, НЕ дублирует: квота/кулдаун инвайта берутся из `domain`.
- Чтение — через read-модели CQRS (запросы `application`), не через агрегаты записи.
