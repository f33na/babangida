# AGENTS.md — `infrastructure`

Перед работой прочитай `../babangida-vault/COMMON.md`. Релевантные ADR: 0009, 0003.

## Роль

Адаптеры к внешнему миру: postgres-репозитории на `sqlx`, реализации доменных
портов. Миграции — через `sqlx-cli`, БД поднимается из flake.

## Границы

- Зависит на `application`, `domain`, `shared`. `domain` про неё не знает.
- Реализует трейты из `domain`, не добавляет в них бизнес-правила.
- SQL явный (без ORM). Интеграционные тесты — против БД из nix.
